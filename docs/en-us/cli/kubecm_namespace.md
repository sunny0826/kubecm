## kubecm namespace

Switch or change namespace interactively

### Synopsis


Switch or change namespace interactively


```
kubecm namespace [flags]
```

![ns](../../static/ns.gif)

### Examples

```

# Switch Namespace interactively
kubecm namespace
# or
kubecm ns
# change to namespace of kube-system
kubecm ns kube-system

```
